How-To Guides

Keyboard Shortcuts
  • Ctrl + Shift + O - Open/close reports panel
  • Ctrl + Shift + R - Generate rating URL
  • Ctrl + Shift + U - Unassign & Close
  • Ctrl + Shift + S - Submit report
  • Ctrl + Shift + X - Submit report, close panel, unassign, close
Using Reports Automator
  1. 1. Make sure you set your name and # of seconds in the fields above. Make sure your name matches (including casing) the one listed in the post-interaction report form
  2. 2. Being in the intercom tab, click the panda icon, then "Open/Close Reports Panel" (or use shortcut Ctrl + Shift + O). This will open the panel for your active conversation. Try it out!
  3. 3. If you select another conversation while the panel is open, it'll update with that conversation's info
  4. 4. Fill the required fields and submit. Real report form will automatically be filled, submitted and closed. You'll get a notification whether the report suceeded or failed.
  5. 5. If you've already submitted a report for a conversation, you'll see a notification about this in the panel (to avoid double entries).
  6. 6. After saving your address and email on this page, it will be saved, but won't visually show up in the input fields again
