Usage Guide

Using Reports Automator

  1. 1. Make sure you set your name and # of seconds in the "Options" tab. Confirm that your name matches (including casing) the one listed in the post-interaction report form
  2. 2. While in the intercom tab, click the panda icon, then "Open/Close Reports Panel" (or use shortcut Ctrl + Shift + O). This will open the panel for your active conversation. Try it out!
  3. 3. Fill the required fields and submit. Real report form will automatically be filled, submitted and closed. You'll get a notification whether the report succeeded or failed.
  4. 4. If you've already submitted a report for a conversation, you'll see a notification about this in the panel (to avoid double entries).
Keyboard Shortcuts
  • Ctrl + Shift + O - Open/close reports panel
  • Ctrl + Shift + R - Generate rating URL
  • Ctrl + Shift + U - Unassign & Close (just closes if already unassigned)
  • Ctrl + Shift + X - Submit report, close panel, unassign, close
  • Ctrl + Shift + M - Unassign
Tips
  1. You can use the following in a note to populate some of the report panel fields:
  2. 1. m: - fills in the minutes field
  3. 2. h: - fills in the hours field
  4. 3. o: - checks the "other" checkbox, and fills in the corresponding text field
  5. 4. Use a semicolon between each if using more than one
  6. Example: m: 2; o: reset Java exercise

Other Extension Functions

Clock In/Out
  1. 1. Make sure you're already logged in to your When I Work and Slack accounts
  2. 2. Click the "Clock In" button before your shift. This will clock in on When I Work and Slack, and open the intercom tab. You'll get a notification whether it succeeded or failed.
  3. 3. The Clock In button will change to "Clock Out"
  4. 4. At the end of your shift, click "Clock Out". This will clock out on When I Work and Slack. You'll get a notification whether it succeeded or failed.
  5. 5. The Clock Out button will change back to "Clock In"
